# Django React Registration System

Практическая работа по разработке клиент-серверного приложения на Django и React.

## Функционал

- Регистрация пользователей
- Авторизация пользователей
- JWT-аутентификация
- Личный кабинет пользователя
- Django REST Framework API

## Backend

- Django
- Django REST Framework
- SQLite

## Frontend

- React
- Vite
- Axios
- React Router

## Запуск Backend

```bash
cd backend
python manage.py runserver
```

## Запуск Frontend

```bash
cd frontend
npm install
npm run dev
```

## Тестирование

```bash
python manage.py test
```
